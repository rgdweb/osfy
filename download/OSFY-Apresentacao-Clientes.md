# 🎯 O que é o OSFY?

O **OSFY** é um sistema completo de gestão desenvolvido especificamente para **assistências técnicas** de celulares, computadores, eletrodomésticos e outros equipamentos.

Uma solução moderna, fácil de usar e acessível de qualquer lugar.

---

# ✨ Funcionalidades Principais

## 📱 1. Gestão de Ordens de Serviço

| Recurso | Descrição |
|---------|-----------|
| **Cadastro de OS** | Registre equipamentos, defeitos, valores e prazos |
| **Código único** | Cada OS tem um código para o cliente acompanhar (ex: A053219) |
| **Status em tempo real** | Recebido → Em diagnóstico → Aguardando aprovação → Em reparo → Concluído |
| **Assinatura digital** | Cliente assina na tela ao retirar o equipamento |
| **Fotos do equipamento** | Registre o estado antes e depois do reparo |
| **Histórico completo** | Todas as alterações ficam registradas |
| **Impressão de OS** | Imprima ou envie por WhatsApp para o cliente |

### 💡 Diferencial:
O cliente pode **acompanhar o status da OS pelo celular** sem precisar ligar na loja!

---

## 👥 2. Gestão de Clientes

| Recurso | Descrição |
|---------|-----------|
| **Cadastro completo** | Nome, telefone, email, CPF, endereço |
| **Histórico de serviços** | Veja todos os atendimentos do cliente |
| **Busca rápida** | Encontre clientes por nome, telefone ou CPF |
| **Agrupamento por OS** | Veja quantas OS cada cliente já fez |

---

## 🛒 3. PDV - Ponto de Venda

| Recurso | Descrição |
|---------|-----------|
| **Venda de produtos** | Venda peças, acessórios, capas, carregadores |
| **Controle de estoque** | Saiba quanto tem de cada produto |
| **Categorias** | Organize produtos por categoria |
| **Histórico de vendas** | Veja todas as vendas realizadas |
| **Controle de caixa** | Abertura e fechamento de caixa diário |
| **Recibo de venda** | Imprima ou envie o recibo para o cliente |

# Comandos Úteis - TecOS

## Para abrir o painel de Publicação

Quando o painel de publicação não aparecer, digite:

```
finalizar projeto
```

Ou em inglês:

```
complete project
```

O assistente irá executar a função `Complete` e o painel aparecerá.
